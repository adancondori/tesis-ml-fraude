#tesis 

Son las respuestas a esas preguntas que nos hemos planteado en los [[Formulación del Problema#Problema General]] y [[Formulación del Problema#Problemas Específicos]]


No está de más mencionar que si al final del estudio se demuestra algo contrario a lo que dice la hipótesis, el estudio igual tiene validez.


## Hipótesis General
Se escribe en presente y de manera afirmativa. Siempre responde a la pregunta que se planteó en el problema general. Es decir si la pregunta dice que si existe relación entre las variables, la respuesta tiene que ser que existe una relación, pero como aún no se sabe qué tipo de relación será, solo se pone que existe esa relación o que existe una relación significativa.


## Hipótesis Específica
Se responden a las mismas preguntas que se plantearon en los problemas específicos. Sí, solo con ellos.

Tipos de Hipótesis:

1. Hipótesis Nula: Asume que no hay diferencias significativas entre las variables que se estudian, se parte entre la suposición que no hay relación entre las variables
2. Hipótesis Alternativa: Es la contraria entre la nula, que afirma la relación entre las variables
3. Hipótesis Correlacional: Establece una relación entre 2 o mas variables, sin establecer una relación causal entre ellas.
4. Hipótesis Causal: Causa-Efecta entre 2 o mas variables
5. Hipótesis Direccionales: Dirección de relación entre 2 o mas variables, por ejemplo a medida que aumenta la variable tal la variable y disminuye.
6. Hipótesis No Direccionales: No establece dirección de la relación entre variables.


Ejemplo de Hipótesis

*Si se propone una estrategia para contribuir a la gestión efectiva del sistema de seguridad de la información en la prevención de intrusos y de acceso no autorizado, se podrá fortalecer la internet/red en la empresa Parpack de Santa Cruz, Bolivia en la gestión 2024.*


El tipo de hipótesis del ejemplo es una hipótesis correlacional-causal, cuando es este tipo de hipótesis debe ser única (se propone primero la propuesta y luego lo que se quiere alcanzar con ella).

Para las hipótesis nula o alternativa se pueden definir varias hipótesis.
