#tesis 

## Técnica
Si se trata de un estudio descriptivo, la técnica es la encuesta o la entrevista. Si se trata de un estudio experimental la técnica es el experimento.

## Instrumento
Es lo que va a utilizar para recoger los datos

Pasos para la creación del instrumento:

Es muy importante revisar la tabla de operacionalización de variables para tener en cuenta las variables con sus respectivas dimensiones y los indicadores de cada una de ellas. También se irán revisando los números de los ítems que correspondan a cada dimensión. para esto vamos a usar el método PIDA:
P: Plantilla (Plantilla para hacer el cuestionario)
I: Items (Items que correspondan a cada indicador)
D: Dimensiones (Cada indicador corresponde a cada indicador)
A: Alternativas (Las alternativas de respuestas siempre deben ponerse para que las personas marquen lo que crean conveniente)

Esta es una plantilla modelo para un cuestionario. La escala de lickert se utiliza para las alternativas de respuestas mayormente es del 1 al 5. Las alternativas jamás deben ser solo dos. Estas deben ser 3 no menos de esto.

## Ejemplo de Cuestionario
![[Pasted image 20251129103842.png]]

Este es un ejemplo de cómo se podría hacer un cuestionario en el que se incluyen las 2 variables. Tanto arriba como abajo se podría incluir esa columna que dice Total de cad dimensión. Eso se hace con la finalidad de facilitar la correción y poder darle al estadístico una data mas clara. Hay que recordar que los objetivos nos piden determinar la relación entre las dimensiones de cada variable. Por eso es mejor cualificar las variables y facilitar así la correción de todos los cuestionarios.

Está demás mencionar que todos los ítems tiene que ser cerrados y con alternativas de respuestas. No puede ser preguntas abiertas porque no habría forma de calificarlos.

Hay que darse cuenta que los ítems no son preguntas. Son ítems y van redactados en primera persona. Pueden ser afirmativos o en negativo, pero es necesario redactar los items de esta manera para que cuando las personas lo lean puedan identificarse no solo con el item sino también con la situación del momento. Así podrían responder mejor cuando el item es una pregunta. Para evitar la sensación de que las personas se sienten evaludas en una pregunta y eso puede generar respuestas falsas.
